## Why greatstorage?

Plain `localStorage` forces you to call `JSON.stringify` and `JSON.parse` everywhere, silently loses `Set`, `Map`, `Date`, and `BigInt` values, and provides no built-in way to expire stale data. greatstorage fixes all of that with a thin, fully typed wrapper that stays out of your way.

<CardGroup cols={2}>
  <Card title="Store anything" icon="box-archive">
    `Set`, `Map`, `Date`, `RegExp`, `BigInt`, circular references — all handled automatically via [devalue](https://github.com/sveltejs/devalue).
  </Card>
  <Card title="TTL / expiration" icon="clock">
    Set a `ttl` in milliseconds or an absolute `expiresAt` timestamp. Expired keys are treated as missing and cleaned up on read.
  </Card>
  <Card title="Namespacing" icon="folder-tree">
    Isolate your keys from third-party scripts with a configurable `prefix`. `clear()` only removes entries in your namespace.
  </Card>
  <Card title="Schema validation" icon="shield-check">
    Validate values on read with any [Standard Schema](https://github.com/standard-schema/standard-schema)-compatible library — Zod, Valibot, ArkType, and more.
  </Card>
  <Card title="Pluggable backends" icon="database">
    Works with `localStorage`, `sessionStorage`, or any `Storage`-compatible implementation, including an in-memory option for tests and SSR.
  </Card>
  <Card title="Tree-shakeable" icon="leaf">
    Dual ESM/CJS build with full TypeScript types. Import from `greatstorage/core` to exclude `devalue` from your bundle when using a custom serializer.
  </Card>
</CardGroup>

## Quick example

```typescript
import { createStorage } from 'greatstorage';

const storage = createStorage({ prefix: 'my-app' });

// Store rich types — no JSON.stringify needed
storage.setItem('tags', new Set(['typescript', 'browser', 'storage']));
storage.getItem('tags'); // Set {'typescript', 'browser', 'storage'}

// Store with TTL
storage.setItem('session-token', 'abc123', { ttl: 60_000 }); // expires in 60s

// Schema-validated reads
import { z } from 'zod';
const UserSchema = z.object({ name: z.string(), age: z.number() });
const user = storage.getItem('user', { schema: UserSchema }); // User | null
```
